# Labyrinth-of-Nosmouth
Text based adventure game where you explore a maze

PROJECT OVERVIEW:
This text-based adventure game is going to be set in an enclosed map, where the player explores a generated maze-like strcture.
The player will use text commands to make decisions. Thngs like movement and actions will be checked for exceptions after each move
to prevent unintended behavior. The game will feature a combat system, and item system, and multiple different enemy types.

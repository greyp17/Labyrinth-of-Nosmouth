# Labyrinth-of-Nosmouth
Text based adventure game where you explore a maze

//Garrison work start
PROJECT OVERVIEW:
This text-based adventure game is going to be set in an enclosed map, where the player explores a generated maze-like structure.
The player will use text commands to make decisions. Things like movement and actions will be checked for exceptions after each move
to prevent unintended behavior. The game will feature a combat system, an item system, and multiple different enemy types.

The game will run in the console and display a text-based interface. It will display a map of the maze and the health of the player to
help them know their current game state.
